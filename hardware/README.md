# Batthub Hardware Files

Upload the schematic and board release files here.

Recommended files:

- `schematic.pdf`: exported schematic for users and reviewers.
- `bom.csv`: bill of materials.
- `pinout.md`: public pinout for SDA, SCL, CE, OTG, INT, VBUS, SYS, BAT, GND, TS/NTC, and ILIM.
- `limits.md`: tested limits for input current, charge current, OTG output, battery type, and temperature.
- CAD source files if the hardware design should be open.

Before publishing, check `docs/HARDWARE_HANDOFF.md`.
